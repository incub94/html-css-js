Тэг создающий переход на следующею строку <br>

Тэги для оформления списков

    Виды списков:

  1. Ненумерованый список
    • Стиль маркера: list-style-type:circle
    • Стиль маркера: list-style-type:square
    • Стиль маркера: list-style-type:none (Пример: <ul style="list-style-type:square">)
  2. Нумерованый список
    1. Стиль маркера: type="A"
    2. Стиль маркера:type="a"
    3. Стиль маркера:type="I"
    4. Стиль маркера:type="i" (Пример: <ol type="I">)
  3. Список определений
    Хто тут?
    Я
    Ты
    Никого
  4. Вложенный список. Примечание: посмотришь как выглядет в пункте 1 и 2)

ССССССЫЫЫЫЫЛКИИИИИ

    Основные правила создания ссылок:

  • Для вставки ссылок испотзуется тэг <a>-</a> c атрибутом href="", текст ссылки вписывается в ""

  • Для добавления относительной ссылки на файл лежащий внутри дочерней/внучатой папки href="folder/file.html или href="folder/folder/file.html соответственно"

  • Для добавления ссылки возвращащей на предыдущую/первую страницу href="../file.html или href="../../file.html соответственно"

  • Что бы при открытии ссылки первичное вкладка не закрывалось, а ссылка окрывалась в новом вкладке исп. атрибут target="_blank"

  • Для открытия эл. почты исп. mailto (Пример: href mailto:bibaboba@mail.ru)

  • И хэшей вам на засыпку

  • Для создания ссылки на определённую часть страницы исп. символ #, для того что бы хэш отправлял нас на нужную часть страницы исп. атрибут id (Пример: href="#biba"; id="biba")

  • Использование картинки как ссылки (Пример: <a href="deepest.html"><img src="corgi.jpg"></a>)
